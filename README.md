# Pushtransm
#### Uses Pushbullet to notify when something has happened.

*Requires a Pushbullet registration*

Basic usage:

> python speedtest.py

Sends data in every 10 mins and saves data to .csv:

> crontab -e

> */10 * * * * /usr/bin/python /home/pi/speedtest/speedtest-cron.sh
